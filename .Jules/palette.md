# Palette's Journal

## 2024-05-22 - Empty States in Chat Interfaces
**Learning:** Users faced with a blank chat interface often experience "writer's block" or confusion about the tool's capabilities. A simple blank screen is a missed opportunity for onboarding.
**Action:** Always implement a friendly "Empty State" or "Welcome" screen that explains the personas (Strategist/Coach) and offers suggested actions. This reinforces the mental model immediately.
